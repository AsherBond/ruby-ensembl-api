Overview

The Ensembl::Core module covers the core databases from ensembldb.ensembl.org and covers mainly sequences and their annotations. For a full description of the database (and therefore the classes that are available), see http://www.ensembl.org/info/software/core/schema/index.html and http://www.ensembl.org/info/software/core/schema/schema_description.html
